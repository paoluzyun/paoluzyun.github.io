# 跑路云

跑路云是一个面向机场优惠信息查询的中文内容站，持续整理公开可见的优惠码、折扣活动、套餐优惠和使用核验说明。

网站关注用户在领取优惠前最需要确认的问题：活动是否仍可查看、优惠入口在哪里、适用范围有哪些、结算页该如何复核，以及优惠码失效时可能是什么原因。内容以清晰、克制和可核验为原则，不编造无法公开验证的代码、价格或有效期。

## 内容范围

- 机场优惠码与公开活动整理
- 新用户、续费、套餐相关优惠说明
- 优惠码使用步骤与结算页核验方法
- 常见失效原因、适用限制和下单前提醒
- 持续更新的文章与活动信息索引
